# wordle clone
play as much wordle as you like!
to decipher the board:
  * correct letter, correct position (green) = [x]
  * correct letter, incorrect position (yellow) = (x)
  * letter is not in word at all (gray) = [ ]

to get started:
copy the wordle.java and 5-letter-words.txt files and run the game in terminal!
