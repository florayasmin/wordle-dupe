# wordle clone
play as much wordle as your heart desires!
to decipher the board:
  * [x] indicates correct letter, correct position (green)
  * (x) indicates correct letter, incorrect position (yellow)
  * [ ] indicates letter is not in word at all (gray)

getting started:
copy the wordle.java and 5-letter-words.txt files and run the game in terminal!
